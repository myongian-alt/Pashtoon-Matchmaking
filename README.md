# Matrimonial App

A starter Expo React Native app for a matrimonial matching experience.

## Getting Started

1. Install dependencies:
   ```bash
   cd matrimonial-app
   npm install
   ```
2. Start the Expo development server:
   ```bash
   npm run start
   ```
3. Open the app on a simulator or physical device.

## Project Structure

- `App.tsx` — app entry point and home screen UI
- `package.json` — project scripts and dependencies
- `app.json` — Expo configuration
- `tsconfig.json` — TypeScript settings
